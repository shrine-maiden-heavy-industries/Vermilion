# Vermilion `xtask`

> [!WARNING]
> This is an internal tool for Vermilion development, Vermilion and the associated libraries are
> located in the [`crates`] directory.

This is a `cargo xtask` runner to help with various automation and maintenance tasks for use with Vermilion development.
