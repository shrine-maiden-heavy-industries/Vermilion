# Introduction

Mercurous is a protocol designed to communicate with HDL simulation engines and runtimes. The goal is to provide a platform and language agnostic way to talk to simulators, as well as debug running simulations and acquire live waveform data.

It was designed for use with [Cinnabar], the HDL model compiler and simulation engine from the [Vermilion] project, and thus this spec also lives alongside them.

## Contributing

The Mercurous specification and it's reference implementation are fully open source and community driven. You can find the source code for [this specification] and the [reference implementation], in the [git repository] on [Codeberg]. Any issues, questions, or comments can be submitted via the [issue tracker], specific information see the [CONTRIBUTING] documentation and maybe open a [pull request]!
