# Vermilion - Temagamite

> [!WARNING]
> This is currently a stub crate for eventual IEEE 1735 support [Vermilion], it is not the focus until
> the main HDL language support is implemented and working well enough to add ancillary support for
> IP encryption/decryption

This crate provides [IEEE 1735] encryption/decryption support [Vermilion], [Cinnabar], and [Alembic].
