# `vermilion-liberty`

> [!WARNING]
> This is currently a stub crate for eventual Liberty support in Vermilion, it is not the focus until the main
> HDL language support is implemented and working well enough to add ancillary support for Liberty files.

Synopsys Liberty library support for Vermilion.
