# Vermilion - Liberty

> [!WARNING]
> This is currently a stub crate for eventual Liberty support in Vermilion, it is not the focus until the main
> HDL language support is implemented and working well enough to add ancillary support for Liberty files.

This crate provides support for [Synopsys Liberty] libraries within [Vermilion] and [Cinnabar].

The following Liberty versions are targeted for support:

- Liberty 2014.09
- Liberty 2015.12
- Liberty 2016.09
- Liberty 2017.06
- Liberty 2018.06
- Liberty 2019.03
- Liberty 2019.12
- Liberty 2020.09
