# Vermilion - SDF

> [!WARNING]
> This is currently a stub crate for eventual SDF support in [Vermilion], it is not the focus until
> the main HDL language support is implemented and working well enough to add ancillary support for
> SDF files.

This crate provides support for [IEEE 1497] SDF (Standard Delay Format) files for [Vermilion].
