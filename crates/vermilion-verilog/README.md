# Vermilion - Verilog

This crate provides [Verilog], [Verilog-AMS], and [SystemVerilog] language support for for [Vermilion] and [Cinnabar].

The following standards are targeted for support:

- Verilog
  - IEEE 1364-1995
  - IEEE 1364-2001
  - IEEE 1364-2005
- Verilog-AMS
  - Accellera Verilog-AMS 2.3.1
  - Accellera Verilog-AMS 2.4
  - Accellera Verilog-AMS 2023
- SystemVerilog
  - IEEE 1800-2005
  - IEEE 1800-2009
  - IEEE 1800-2012
  - IEEE 1800-2017
  - IEEE 1800-2023
