# Vermilion - VHDL

This crate provides  [VHDL], [VHDL-AMS], and partial [BSDL] language support for for [Vermilion] and [Cinnabar].

The following standards are targeted for support:

- VHDL
  - IEEE 1076-1987
  - IEEE 1076-1993
  - IEEE 1076-2000
  - IEEE 1076-2002
  - IEC 61691-1-1:2004
  - IEEE 1076-2007
  - IEEE 1076-2008
  - IEC 61691-1-1:2011
  - IEEE 1076-2019
  - IEC 61691-1-1:2023
- VHDL-AMS
  - IEEE 1076.1-1999
  - IEEE 1076.1-2007
  - IEC 61691-6:2009
  - IEEE 1076.1-2017
  - IEC 61691-6:2021
- BSDL
  - IEEE 1149.1-2013
