# Mercurous

> [!WARNING]
> This is currently unstable, a draft, and not the primary focus until Vermilion has proper HDL support.

Mercurous is an HDL agnostic simulator control and transport protocol used in the [Vermilion] [Cinnabar] model compiler and simulator.
