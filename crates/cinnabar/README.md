# Cinnabar

> [!WARNING]
> Cinnabar is a wishlist part of Vermilion, it's not going to get any focus until the core of Vermilion
> and all of it's features are properly implemented and working well.

Cinnabar is an HDL model compiler and simulation engine that is part of the [Vermilion] project.
